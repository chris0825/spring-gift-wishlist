## 제약 조건
- [자바 코드 컨벤션](https://google.github.io/styleguide/javaguide.html)을 지키면서 프로그래밍 한다

- indent(인덴트, 들여쓰기) depth를 2이하로 구현한다
- 3항 연사자를 사용하지 않는다
- 함수(method)의 최대 라인수는 15이다
- else 예약어를 사용하지 않는다(switch/case도 사용 불가)
